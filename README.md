## Dotfiles

Dotfiles script for personal use. Based on [Michael Jackson](https://github.com/mjackson/dotfiles) dotfiles structure.

## Clone Repository

```bash
git clone https://github.com/arianbessonart/dotfiles.git ~/dotfiles
```

## Install

```bash
cd ~/dotfiles
chmod +x setup*
./setup
```
